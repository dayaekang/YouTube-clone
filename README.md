## 프로젝트 실행 방법
```
  yarn start
```
<br>

## 사용한 기술
- Tailwind css
- Context API
- Youtube API
- axios 라이브러리

## 주요 기능

#### Youtube API를 호출해 
- 실시간 인기 동영상 받아오기
- 동영상 검색하기
- 썸네일 받아오기
- 동영상 클릭시 관련된 비디오 받아오기

#### 동영상을 클릭하면 해당영상의 세부정보 보여주기

#### 영상이 등록된 시간을 현재 시간 기준으로 time ago 보여주기
